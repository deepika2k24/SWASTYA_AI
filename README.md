*SWASTYA_AI* is an AI-powered pre-diagnostic healthcare assistant. It helps users quickly understand possible health conditions based on their symptoms, suggests basic remedies, and guides them toward professional medical advice‚Äîall before visiting a doctor. Using **Naive Bayes** and **NLP (spaCy)**, the system predicts top probable diseases from user-entered symptoms.

---

## üåü Features

- Natural language symptom input and extraction using **spaCy NLP**  
- Prediction of **top 3 probable diseases**  
- Lists associated symptoms for each disease  
- User-friendly interface built with **HTML, CSS, and JS**  
- Backend powered by **Flask** and a **Naive Bayes** model
